# Release 2.0-R1 — Operational Product Foundation

R1 creates the Version 2 composition contract without changing the live
application entry point.

The release establishes:

- eight Staffology-aligned product sections;
- an operational workspace registry;
- capability-aware workspace metadata;
- availability states;
- deterministic navigation composition;
- public frontend exports.

Existing Staffology workspaces remain the implementation assets. Later R1
integration work can consume this registry without duplicating payroll
behaviour.
