# Release 1.2 RC2-E1

## Staffology Regular Pay Workspace

RC2-E1 activates the Staffology Pay Options workspace and connects
its first subsection: Regular Pay.

## Scope

- Pay Options subsection navigation
- Regular Pay adapter
- Regular Pay read-only panel
- Staffology contract-driven field resolution
- Currency and hours formatting
- Reserved subsection states
- Modular verification

## Regular Pay fields

- Employment type
- Pay basis
- Pay frequency
- Payroll frequency
- Annual salary
- Regular pay
- Hourly rate
- Daily rate
- Working pattern
- Normal hours
- Normal days
- Payroll code

## Boundary

RC2-E1 introduces no salary amendment, payroll update or Staffology
write operation.
