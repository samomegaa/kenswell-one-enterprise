# Release 1.2 RC2-E6

## Staffology Benefits Workspace

This release activates the final Pay Options subsection.

## Runtime states

- unavailable: no recognised benefits collection exists;
- empty: a recognised collection exists but contains no items;
- populated: benefit records appear in a read-only table.

## Contract rules

- Missing values display as an em dash.
- Boolean values display only when supplied.
- Currency values use GBP formatting.
- No values are calculated or inferred.
- No write operation is introduced.
