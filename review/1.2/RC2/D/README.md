# Release 1.2 RC2-D

## Staffology API Discovery and Contract Catalogue

RC2-D establishes Staffology as the reference payroll API contract
for the current Kenswell One payroll platform.

## Purpose

The release catalogues:

- Staffology API operations already used by Kenswell One
- Kenswell API routes that expose Staffology data
- Response structures
- Employee workspace sections
- Mapped fields
- Unmapped fields
- Read and write capabilities
- Authentication and security boundaries
- Personally identifiable information classifications

## Workspace domains

- Employers
- Employees
- Basic Details
- Employment
- Pay Options
- Attachment Orders
- Bank Account
- Leave
- Pension
- Notes
- More

## Release boundary

RC2-D introduces no production UI changes and no Staffology
mutations.

It creates the evidence and contract catalogue required for the
remaining Staffology workspace releases.
