# Release 1.2 RC2-C1

## Staffology Basic Details

RC2-C1 introduces the first Staffology-aligned employee workspace
section.

The release:

- preserves the Staffology employee response;
- resolves basic employee details without creating a generic provider model;
- renders a read-only Basic Details workspace;
- uses the Staffology primary tab structure;
- keeps unavailable sections visible for subsequent releases;
- introduces no Staffology write operations.

## Basic Details scope

- Employee name
- Title
- First name
- Middle name
- Last name
- Date of birth
- Gender
- Marital status
- Email
- Alternative email
- Telephone
- Mobile
- National Insurance number
- Previous surname
- Passport number
- Address
- Staffology employee reference
