# Release 1.2 RC2-E4.1

## Staffology Tax & NI Contract Alignment

- boolean tax basis `false` displays as `Cumulative`;
- boolean tax basis `true` displays as `Week 1 / Month 1`;
- statutory booleans display as `Yes` or `No`;
- recognised student-loan identifiers receive payroll-friendly labels;
- currency values use GBP formatting;
- missing values remain missing and render as an em dash.

The runtime contract remains unchanged and read-only.
