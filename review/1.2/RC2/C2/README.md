# Release 1.2 RC2-C2

## Staffology Employment Workspace

RC2-C2 connects the Staffology-aligned Employment tab to the
live employee response.

## Scope

- Employment status
- Job title
- Workplace postcode
- Start date
- Continuous start date
- Payroll code
- Starter declaration
- Payroll ID change handling
- On-hold status
- Director status
- Directorship dates
- National Insurance calculation method
- Right-to-work status
- Freeport status
- Investment Zone status
- Veteran status
- Employee-left status
- Overseas secondment
- Registered pension income
- CIS subcontractor
- Apprentice status
- Furlough status

## Release boundary

The Employment workspace is read-only.

No Staffology employee amendment or payroll mutation is introduced
by RC2-C2.
