# Release 1.2 RC2-E5

## Staffology Other Pay Options Workspace

This release activates the Other subsection within Staffology Pay Options.

## Sections

- Payroll controls
- Working arrangement
- Reporting indicators

## Rules

- The workspace is read-only.
- Exact aliases take priority.
- Pay and employment paths are preferred.
- Boolean values are presented only when supplied.
- Missing values display as an em dash.
- No values are inferred or calculated.
