# Release 1.2 RC2-E2

## Staffology Additions & Deductions Workspace

RC2-E2 connects the Additions & Deductions Pay Options subsection to
the Staffology employee runtime.

## Scope

- Addition and deduction contracts
- Collection discovery
- Read-only additions presentation
- Read-only deductions presentation
- Empty and unavailable states
- Pay Options subsection integration
- Modular verification

## Boundary

No Staffology write operation is introduced.
