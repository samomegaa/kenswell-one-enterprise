# Release 1.2 RC2-C2.1

## Staffology Employment Contract Alignment

C2.1 aligns the Employment adapter with the live Staffology
employee contract.

## Corrections

- Search the complete Staffology employee response for recognised
  employment properties.
- Preserve explicit false values.
- Do not convert unresolved booleans into false.
- Display unavailable values as an em dash.
- Resolve Staffology director and starter fields across nested
  response structures.
- Preserve the read-only workspace boundary.

## Target fields

- Start date
- Continuous start date
- Starter declaration
- Workplace postcode
- Payroll ID change
- Director status
- Start of directorship
- End of directorship
- Alternative NI method

No Staffology write operation is introduced.
