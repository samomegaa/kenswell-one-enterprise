# Release 1.2 RC2-E1.1

## Staffology Regular Pay Contract Alignment

E1.1 aligns the Regular Pay workspace with Staffology's actual
field names and payroll terminology.

## Target fields

- Schedule
- Basis
- Working Pattern
- Monthly Amount
- Annual Salary
- Pay Code
- Pro-rata Adjustments
- Base Hourly Rate
- Base Daily Rate

## Contract rules

- Exact Staffology aliases take priority.
- Broad aliases that could match unrelated employee properties
  are not permitted.
- Missing values remain unavailable and display as an em dash.
- No values are calculated or inferred.
- The workspace remains read-only.
