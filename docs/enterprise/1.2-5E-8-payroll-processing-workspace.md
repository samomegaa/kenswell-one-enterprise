# Release 1.2 — 5E-8

## Enterprise Payroll Processing Workspace

This release introduces the operational payroll-run workspace.

### Workflow

1. Select pay schedule and period.
2. Select employees.
3. Create the payroll run.
4. Validate employee and payroll readiness.
5. Generate gross-to-net preview.
6. Approve the versioned run.
7. Process payroll.

### API contract expected

```text
POST /api/payroll-runs
POST /api/payroll-runs/:runId/validate
GET  /api/payroll-runs/:runId/preview
POST /api/payroll-runs/:runId/approve
POST /api/payroll-runs/:runId/process
```

The workspace is intentionally separated from provider submission and RTI.
Those capabilities follow in later payroll releases.
