# Enterprise 1.0-GA-A — Enterprise Repository Framework

## Status

## Purpose

This release introduces the Enterprise Repository Framework.

It provides a consistent persistence abstraction for all future Kenswell One modules.

## Capabilities

- Repository contracts
- Repository types
- Repository registry
- Repository provider
- Persistence session
- Repository middleware
- Core export integration
- Verification-backed repository behaviour
