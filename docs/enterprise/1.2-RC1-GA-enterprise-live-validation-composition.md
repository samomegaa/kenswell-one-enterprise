# Version 1.2 RC1-GA

## Enterprise Live Validation Composition

RC1-GA connects the RC1-G validation framework to the real Enterprise
implementation.

The composition uses:

- `StaffologyClient`;
- `StaffologyPayrollProvider`;
- the Staffology employee mapper;
- `EmployeeWorkspaceService`;
- the RC1-G validation runner.

A small adapter bridges the real workspace service method
`get(employeeId)` to the validation framework method
`getWorkspace({ employeeId })`.

The live employee service performs a read-only Staffology request:

```text
GET /employers/{employerId}/employees/{employeeId}
```

No provider mutation is performed. Credentials are read only from environment
variables and are never included in the generated report.
