## Security Headers

Default headers include:

- `x-content-type-options`
- `x-frame-options`
- `referrer-policy`
- `x-xss-protection`
- `cross-origin-opener-policy`
- `cross-origin-resource-policy`
- `permissions-policy`
- `strict-transport-security`

## Runtime Guard Model

Runtime guards evaluate runtime/environment-level security decisions.

Examples:

- environment allowed
- maintenance mode
- licence active
- tenant enabled
- platform lockdown

## Request Guard Model

Request guards evaluate request-level security decisions.

Examples:

- HTTP method allowed
- header accepted
- origin allowed
- content type allowed
- request size allowed
