# Version 1.2 RC1-A

## Enterprise Workspace API & Adapter

RC1-A introduces the integration boundary between the Enterprise Payroll
Workspace and the React Tax & Payroll product.

The release defines:

```text
GET /api/employees/:employeeId/workspace
```

The endpoint returns one authoritative payload containing the employee,
workspace schema, visible workspace, readiness dashboard, provider panel,
contract identity and integration metadata.

The React application receives the payload through:

- `getEmployeeWorkspace()`
- `adaptEmployeeWorkspace()`
- `useEnterpriseEmployeeWorkspace()`

The Enterprise layer owns payroll structure, visibility, readiness and
provider state. React owns presentation and local user interaction.
