# Version 1.2 RC1-C

## Dynamic React Field Renderer

RC1-C replaces the temporary section summaries introduced in RC1-B with a
generic React renderer driven by Enterprise payroll field metadata.

Supported controls:

- text;
- email;
- telephone;
- textarea;
- number;
- date;
- datetime;
- checkbox;
- select;
- currency;
- percentage;
- read-only;
- JSON.

The renderer resolves values through each field's `providerBinding`. It also
surfaces help text, readiness requirements, read-only state and validation
messages.

RC1-C is read-only in the integrated workspace. Local editing and immutable
draft updates are introduced in RC1-E.
