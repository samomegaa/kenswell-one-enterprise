# Version 1.2 RC1-F

## Provider Synchronisation Panel UI

RC1-F exposes the provider synchronisation model created in Release 5E-9A5
inside the canonical Employee Workspace.

The panel renders:

- provider and connection summary;
- synchronisation status;
- diagnostics;
- synchronisation history;
- provider conflicts;
- direct navigation to affected sections;
- refresh and prepare-synchronisation actions.

The panel is provider-neutral. It consumes the immutable provider panel
returned by the Enterprise Workspace API.

Prepare Synchronisation currently creates a local operation summary only.
The live provider mutation boundary is deliberately deferred to RC1-G.
