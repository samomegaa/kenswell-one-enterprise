# Enterprise 1.0-GA-B — Enterprise Scheduler & Background Jobs

## Status

Completed.

## Purpose

This release introduces the Enterprise Scheduler and Background Jobs framework.

It provides a consistent scheduling and background execution abstraction for all future Kenswell One platform services and business modules.

## Capabilities

- Job contracts
- Job types
- Job registry
- Scheduler engine
- Manual job execution
- Scheduled job state tracking
- Failed job tracking
- Cancellation support
- Scheduler middleware
- Core export integration

## Job Types

- once
- recurring
- delayed
- startup
- manual

## Job States

- registered
- scheduled
- running
- completed
- failed
- cancelled
