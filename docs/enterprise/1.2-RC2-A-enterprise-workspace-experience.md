# Release 1.2 RC2-A

## Enterprise Workspace Experience

RC2-A replaces the temporary raw JSON workspace evidence introduced
during H4C with the established Kenswell Employee Workspace framework.

The dynamic workspace now uses:

- `WorkspaceLayout`;
- `WorkspaceHeader`;
- `WorkspaceNavigation`;
- Employee Summary;
- Employment;
- Payroll;
- Tax and NI;
- Pension;
- Leave;
- Provider;
- Timeline;
- Audit.

A runtime adapter projects Staffology and Enterprise workspace payloads
into the canonical employee view model expected by the existing
workspace panels.

The adapter preserves the runtime contract:

```js
{
  employer,
  employee,
  workspace
}
```

No backend endpoint changes are required.

The existing linked-client Provider Centre workflow remains intact.
Only the Enterprise runtime workspace presentation changes.
