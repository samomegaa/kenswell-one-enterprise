# Release 1.0-B4 — Enterprise Context

## Status

Completed.

## Purpose

This release introduces the Enterprise Context layer for Kenswell One Enterprise 1.0.

The context layer provides a single immutable request-scoped object containing:

- request identity
- correlation identity
- tenant identity
- user identity
- authentication metadata
- RBAC roles and permissions
- runtime metadata
- feature flags
- lifecycle metadata
