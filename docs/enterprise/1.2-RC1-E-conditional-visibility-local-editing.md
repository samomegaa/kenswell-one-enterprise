# Version 1.2 RC1-E

## Conditional Visibility & Local Editing

RC1-E introduces an immutable local employee draft while retaining the
Enterprise workspace payload as the source of truth.

The release provides:

- editable metadata controls;
- immutable nested draft updates;
- dirty-field and dirty-section tracking;
- metadata-driven conditional visibility;
- discard support;
- changed-field save payload generation.

The Prepare Save action intentionally logs the immutable change payload.
Persistence is connected only after the Enterprise update boundary and
provider synchronisation workflow are available.
