# Enterprise 1.0-GA-D — Enterprise Search & Indexing

## Status

Completed.

## Purpose

This release introduces the Enterprise Search and Indexing Framework.

It provides a consistent platform-wide abstraction for indexing and searching enterprise records.

## Capabilities

- Search document contracts
- Search document types
- Search index status tracking
- Search index registry
- Search provider
- In-memory search abstraction
- Type-filtered search
- Removed document exclusion
- Search middleware
- Core export integration

## Search Document Types

- entity
- read_model
- audit_event
- notification
- job
- file
- document

## Search Index Statuses

- created
- indexed
- removed
- failed
