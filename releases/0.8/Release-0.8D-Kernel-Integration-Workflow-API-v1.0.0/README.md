# Release 0.8-D — Kernel Integration + Workflow API

This release integrates the Workflow Engine into the Kenswell One platform kernel and exposes workflow orchestration through API routes.

## Capabilities

- Registers `workflowService` inside the Platform Kernel.
- Connects workflow events to the Event Bus.
- Adds REST API routes for workflow lifecycle operations.
- Provides a repeatable integration guide and smoke test plan.
